YARN

yarn est un gestionnaire de paquets au même titre que npm
Il a l'avantage d'être plus rapide que npm pour installer les dependances.

pour passer de npm a yarn dans un projet :

npm install -g yarn

il me suffit ensuite de taper :yarn pour installer mes dépendances lorsqu'on installe les dependances pour la premiere fois cela va générer un fichier yarn.lock

celui-ci est une représentation de toutes les dependances dont il a besoin avec les versions ainsi qu'un chemin pour télécharger ces versions compressées donc il n'a pas besoin d'aller sur git ou autre. Si vous essayez d'accéder à l'une de ces url vous téléchargerez automatiquement un fichier tar.gz

Ce fichier la doit donc être versionner pour les autres personnes qui vont installer le projet, les dépendances seront générées a partir de ce fichier lockfile.
